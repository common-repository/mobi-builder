This plugin allows you to embed a Mobi Builder project into your website.

== Description ==

Mobi Builder is an online website that allows you to build mobile apps without coding. This plugin allows you to embed public Mobi Builder projects into your website.

== Frequently Asked Questions ==

= How do I use this plugin? =

We have a guide on how to use this plugin at the following URL: https://mobibuilder.tech/documentation/public-projects/

= What customizable options are there for this plugin? =

You can change whether you want a 'Open this project in a new window' button to appear and various options about the border surrouding the embedded project.
